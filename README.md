# Veiled Transform

Veiled Transform is a browser based image transformation tool that turns normal uploaded images into mysterious dark fantasy styled artwork inspired by creature aesthetics.

## Live Preview

Upload an image, choose a style, generate your transformed result, then download or share it instantly.

## Features

- Drag and drop image upload
- Supports PNG, JPG, WEBP, GIF
- Multiple Veiled styles
- Fast in browser rendering
- Download transformed image as PNG
- Share on X
- Responsive mobile friendly design
- Animated immersive interface

## Styles Included

- Mosaic Veil
- Shadow Bloom
- Fractal Eye
- Ember Mask
- Void Skull
- Glitch Entity

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Canvas API

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/veiled-transform.git
cd veiled-transform
python -m http.server
```

Open in browser:

http://localhost:8000
